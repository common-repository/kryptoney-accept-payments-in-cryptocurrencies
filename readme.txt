Accept Payments in Cryptocurrencies

== Description ==

This plugin is developed to provide checkout experience with Kryptoney where you can let customer to checkout in Cryptocurrencies and receive payment in either in your crypto wallets or in FIAT currencies. This plugin is compatiable with woocommerce.

Features:

* Debug mode
* Instant Payment Notification
* Restrict cryptocurrencies

Sign up here: https://www.kryptoney.com/register

== Installation ==

= Automatic installation: =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don't need to leave your web browser. To do an automatic install, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type "Kryptoney" and click Search Plugins. Once you've found our plugin you can view details about it such as the the rating and description. Most importantly, of course, you can install it by simply clicking Install Now.

= Manual Installation: =

Unzip the files and upload the folder into your plugins folder (/wp-content/plugins/) overwriting previous versions if they exist
Activate the plugin in your WordPress admin area.
configuration

= Easy steps to install the plugin: =

Upload "woocommerce-kryptoney" folder/directory to the /wp-content/plugins/ directory
Activate the plugin through the 'Plugins' menu in WordPress.
Go to WooCommerce=> Settings=> Checkout=> Kryptoney.

= Setup

For setting up Kryptoney, you need an API key and a Secret key credentials of your Kryptoney account.

API and Secret key can be obtained from Kryptoney backend's setting page: https://kryptoney.com/app/account/update. Follow these steps to get the credentials.

1. Register and create an account at Kryptoney.com (https://kryptoney.com/register)
2. Goto Setting => Integration
3. Copy API and Secret key
4. Go to WooCommerce => Settings => Checkout=> Kryptoney
5. Enter credentials in relevant fields
6. Save
